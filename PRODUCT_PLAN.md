# 北邮小镇 · 产品策划企划案

> **版本**: v0.2 草案  
> **日期**: 2026-07-08  
> **定位**: 校园元宇宙 · AI 驱动的虚拟校园社交探索平台

---

## 一、项目愿景

**让每一位北邮同学，都能在虚拟校园里"遇见"真实的彼此。**

"北邮小镇"不是传统的管理系统或信息门户。它是一个**有温度的虚拟校园**——在这里，你可以走到体育馆约球、去麦当劳拼单、在图书馆偶遇同学、在公告栏发现新鲜事。AI 智能体让每个 NPC 像真人一样对话，双人系统让真正的社交从"偶遇"开始。

### 2.4 信息面板系统 📋

## 三、未来功能规划 —"玩起来"

> 以下功能按优先级分为三个层级。P0 为下一阶段立即实施，P1 为半年内规划，P2 为远期愿景。

---

### 🎯 P0 — 核心玩法（让用户"留下来"）

#### 3.1 每日任务 + 积分系统

```
┌─────────────────────────────────────┐
│  📅 今日任务            ⭐ 积分: 120  │
│                                     │
│  ✅ 登录小镇              +10 ⭐     │
│  ✅ 与一位NPC对话          +15 ⭐     │
│  ⬜ 在二手市场发布一件商品   +30 ⭐    │
│  ⬜ 与另一位同学打招呼      +25 ⭐    │
│  ⬜ 进入3个不同建筑         +20 ⭐    │
│  ⬜ 完成一次AI对话          +20 ⭐    │
│                                     │
│  🎁 连续7天打卡额外奖励: +50 ⭐      │
└─────────────────────────────────────┘
```

**设计要点**：
- 每日 4-6 个任务，覆盖探索/社交/交易/对话四类行为
- 任务引导用户自然体验所有功能模块
- 积分可兑换虚拟物品（头像框、皮肤、特效）


#### 3.2 成就系统

| 成就 | 条件 | 奖励 |
|------|------|------|
| 🗺️ 探索者 | 进入所有 9 个建筑 | 金色地图勋章 |
| 💬 话痨 | 累计 AI 对话 50 次 | 闪电聊天气泡 |
| 🤝 社交达人 | 加 10 个好友 | 社交达人称号 |
| 🛒 小商贩 | 发布 5 件二手商品 | 店铺装修模板 |
| 🏆 连续打卡王 | 连续 30 天登录 | 专属皮肤 |
| 🎓 学霸 | 在图书馆/机房/科研楼累计停留 10 小时 | 学霸光环特效 |

**设计要点**：
- 成就分为隐藏成就和可见成就，增加探索乐趣
- 成就系统天然制造"收集欲"和"比较欲"


#### 3.3 好友系统深度化

现有双人"偶遇" → 升级为真实好友系统：

```
加好友流程：
  校园偶遇 → 弹窗点"加好友" → 对方收到好友申请 → 同意 → 成为好友

好友功能：
  📍 查看好友实时位置（校园地图上显示好友图标）
  💬 一对一私聊（文字 + 表情）
  🎁 赠送礼物（用积分兑换的虚拟礼物）
  🏃 传送到好友身边（消耗积分）
  👀 查看好友资料卡（成就 / 积分 / 足迹）
```

**设计要点**：
- 实时位置共享是核心"玩点"——"你在哪？我来找你！"
- 礼物系统制造人情往来



#### 3.4 二手市场真实化

现有商品卡片 → 升级为完整交易系统：

```
发布商品：
  📸 上传图片
  📝 填写标题 / 价格 / 成色 / 描述
  🏷️ 选择分类标签

浏览与筛选：
  🔍 关键词搜索
  🏷️ 按标签筛选（教材 / 数码 / 外设 / 生活 / 出行 / 考研）
  📊 按价格 / 时间排序

交易流程：
  买家点击"联系" → 私聊窗口 → 议价 → 线下交易
  交易完成后可互评（好评/中评/差评）
  卖家信用分展示
```

**设计要点**：
- 二手市场是大学生"刚需"，天然驱动 DAU（日活）
- 信用评价体系防止欺诈

#### 3.6 校园排行榜

| 排行榜 | 维度 | 刷新频率 |
|--------|------|----------|
| 🏆 积分榜 | 本周/本月/总积分 | 每小时 |
| 💬 社交榜 | 好友数量 + 互动次数 | 每天 |
| 🛒 交易榜 | 成交单数 + 好评率 | 每天 |
| 🔥 活跃榜 | 登录天数 + 在线时长 | 每周 |
| ⭐ 贡献榜 | 发起活动数 + 参与数 | 每周 |

**设计要点**：
- 排行榜制造"比较"和"炫耀"心理
- 周榜 / 月榜给予追赶动力
- Top 玩家获得特殊标识（名字颜色 / 头像框）

### 🚀 P2 — 生态闭环（让用户"离不开"）

#### 3.8 AI 个人助手

每位同学拥有专属 AI 助手：
- 📅 **日程提醒**：课程表导入、考试倒计时、活动提醒
- 📖 **学业问答**：培养方案解读、选课建议、考研规划
- 🗺️ **校园导航**：语音指引去往任意地点
- 💡 **智能推荐**：根据兴趣推荐活动 / 商品 / 好友


#### 3.9 课程 + 学习场景

```
机房 / 图书馆 / 科研楼 → 学习模式入口

功能：
  📚 教学楼课程表实时查询（空教室查找）
  💻 机房空闲机器实时显示
  📖 图书馆座位预约
  👥 同班同学学习小组（按课程自动建群）
  📝 校园笔记分享（二手笔记交易）
  🎓 考研专区（资料共享 + 研友匹配）
```


#### 3.10 社团 + 组织入驻

```
社团可在小镇申请"社团之家"：

🏠 社团专属空间（可装扮）
📢 社团活动发布（自动推送全校）
👥 社团成员管理
📊 社团活跃度排行
🎯 社团任务系统（完成社团任务得双倍积分）
```


#### 3.11 赛季制活动

| 赛季 | 时间 | 主题 | 特殊玩法 |
|------|------|------|----------|
| 🌸 迎新季 | 9 月 | 新生入学 | 学长学姐带新生游览校园得积分 |
| 🎃 万圣节 | 10 月 | 校园寻宝 | 地图上随机刷新糖果，集齐换限定皮肤 |
| ❄️ 考试季 | 1 月 | 学习冲刺 | 在图书馆/机房学习时长翻倍积分 |
| 🌸 毕业季 | 6 月 | 青春留念 | 二手集市特别活动，拍照打卡 |


#### 3.12 更多 AI NPC + 剧情线

现有 3 个 NPC → 扩展为 15+ 个：

| NPC | 位置 | 功能 |
|-----|------|------|
| 宿舍阿姨 | 宿舍 | 生活琐事问答、失物招领 |
| 机房管理员 | 机房 | 设备报修、打印服务 |
| 科研导师 | 科研楼 | 大创/竞赛咨询 |
| 图书馆员 | 图书馆 | 借阅查询、座位预约 |
| 事务大厅老师 | 事务大厅 | 学生证补办、成绩单打印 |
| 图书管理员 | 图书馆 | 图书推荐、借阅提醒 |

每个 NPC 有独特的"故事线"——多次对话后解锁隐藏剧情。

## 四、商业模式构想（远期）

| 模式 | 说明 |
|------|------|
| 🏪 **校园 O2O** | 麦当劳等周边商家入驻，线上点单线下核销，平台抽佣 |
| 📢 **精准广告** | 培训机构/考研机构按兴趣标签投放，学生屏蔽可控 |
| 🎨 **虚拟物品** | 限定皮肤/装扮付费（主力收入），基础功能完全免费 |
| 🛒 **交易抽成** | 二手市场大额交易（>500元）收 1% 手续费 |
| 👔 **企业招聘** | 毕业季企业入驻，精准推送岗位给相关专业毕业生 |

> ⚠️ 核心原则：**基础功能永远免费**。付费仅限于虚拟装扮和增值服务。

## 五、技术架构展望

```
┌─────────────────────────────────────────┐
│            前端 (Phaser 3 + Vite)         │
│  ┌─────────┐ ┌──────┐ ┌──────────────┐  │
│  │ 校园地图 │ │ 场景 │ │ UI面板/弹窗    │  │
│  │ 双人系统 │ │ 对话 │ │ 侧边栏/导航    │  │
│  └─────────┘ └──────┘ └──────────────┘  │
└──────────────────┬──────────────────────┘
                   │ REST + WebSocket
┌──────────────────▼──────────────────────┐
│          后端 (FastAPI → Spring Boot)     │
│  ┌─────────┐ ┌──────┐ ┌──────────────┐  │
│  │ 用户系统 │ │ 社交 │ │ 交易/积分      │  │
│  │ 智能体   │ │ 消息 │ │ 数据分析       │  │
│  └─────────┘ └──────┘ └──────────────┘  │
└──────────────────┬──────────────────────┘
                   │
┌──────────────────▼──────────────────────┐
│           数据与 AI 层                    │
│  ┌─────────┐ ┌──────┐ ┌──────────────┐  │
│  │ PostgreSQL│ │Redis │ │ LLM (DeepSeek) │  │
│  │ 用户数据  │ │ 缓存  │ │ 智能体大脑     │  │
│  └─────────┘ └──────┘ └──────────────┘  │
└─────────────────────────────────────────┘
```

**关键升级**：
- SQLite → PostgreSQL（支持高并发）
- 引入 WebSocket（实时位置共享、私聊、活动通知）
- 引入 Redis（排行榜缓存、会话管理）
- LLM 智能体升级（RAG 接入校园知识库）

## 六、总结 — 为什么同学会"玩起来"？

| 驱动力 | 对应功能 |
|--------|---------|
| 🔍 **探索欲** | 9 个建筑 + 隐藏成就 + NPC 剧情线 |
| 🤝 **社交需求** | 双人偶遇 + 好友实时位置 + 活动组局 |
| 💰 **实用价值** | 二手交易 + 课程查询 + 座位预约 |
| 🏆 **成就感** | 排行榜 + 积分系统 + 成就徽章 |
| 🎨 **自我表达** | 装扮系统 + 宿舍装修 + 称号展示 |
| 📅 **日常习惯** | 每日任务 + 连续打卡 + 赛季活动 |
| 🎉 **新鲜感** | 赛季活动 + 限定皮肤 + 新 NPC 上线 |

**一句话总结**：北邮小镇不是工具，是**校园生活的数字镜像**——让虚拟与现实互相滋养，让每一次打开都有理由。
